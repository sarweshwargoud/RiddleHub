# AWS Riddle

This repository is designed to showcase various riddles that can be used to enhance problem-solving and critical-thinking skills. The riddles are organized by difficulty and topic, making it easy to find a riddle that suits your needs.

## Getting Started

To get started with the riddles in this repository, simply navigate to the `riddles` directory and choose a riddle that you're interested in. Each riddle includes a brief description, the challenge itself, and the solution.

## Contributing

If you would like to contribute your own riddles, feel free to submit a pull request! Please ensure that your riddles follow the structure and style of the existing entries.
